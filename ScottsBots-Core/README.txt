To install this API do the following:

1. Run ant from root.
2. Copy the files from /lib and /dist to your CLASSPATH.

The following Native Packages ARE REQUIRED:
1. FreeTTS
2. Java Communications API

Optional Packages (Windows)
3. QuadmoreTTS.dll - Microsoft Text To Speech
4. QuadmoreSR.dll - Microsoft Speech Recognition
5. Java Media Framework 2.1

Optional Packages (UBuntu)
6. Festival & Foxvest apt-get packages. 
7. http://code.google.com/p/v4l4j/ 

That's it, enjoy!
